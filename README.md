## Employee Retention Prediction

## Project Overview 
This project aims to develop a machine learning model to predict whrthwer data scientists are likely to seek job changes , aiding in talent retention and recruitment planning. It should help in optimizing retention strategies, proactive recruitment, and better resource allocation using a predictive model, comprehensive data visualizations, and a comparative analysis of various machine learning algorithms.
